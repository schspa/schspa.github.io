光标控制

These are not described in ECMA-48.  We list the Set Mode
sequences; the Reset Mode sequences are obtained by replacing the
final 'h' by 'l'.
ESC [ ? 25 h
      DECTECM (default on): Make cursor visible.

  • 显示光标

    echo -e "\033[?25h"

  • 隐藏光标

    echo -e "\033[?25l"
